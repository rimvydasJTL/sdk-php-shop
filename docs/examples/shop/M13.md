# (M13) Download an attachment

```php
use Mirakl\MMP\Shop\Client\ShopApiClient;
use Mirakl\MMP\Shop\Request\Inbox\DownloadInboxThreadsRequest;

$api = new ShopApiClient('API_URL', 'API_KEY', 'SHOP_ID');
$request = new DownloadInboxThreadsRequest('ATTACHMENT_ID');
$result = $api->downloadInboxThreads($request);
// $result => @see \Mirakl\Core\Domain\FileWrapper
```
