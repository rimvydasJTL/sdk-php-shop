# (RO02) List shop roles

```php
use Mirakl\MMP\Shop\Client\ShopApiClient;
use Mirakl\MMP\Shop\Request\User\Shop\GetUsersShopsRolesRequest;

$api = new ShopApiClient('API_URL', 'API_KEY', 'SHOP_ID');

$request = new GetUsersShopsRolesRequest();
$result = $api->getUsersShopsRoles($request);
// $result => @see \Mirakl\MMP\Common\Domain\Collection\User\Shop\RoleCollection

foreach ($result as $role) {
    $role->toArray();
}
```
